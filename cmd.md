# Command cmd

* Directory change 

```sh
cd c:\<folder_name>
```

* Display the current directories UNIX
```sh
pwd
```

* List the current directories WINDOWS
```sh
dir
```

* List the current directories UNIX
```sh
ls
```

* Delete file in WINDOWS
```sh
del <filename>
```

* Delete file in UNIX
```sh
rm <filename>
```
